### 如何使用：
命令行进入该目录->
- WallePackage.py -h  查看帮助
- WallePackage.py -c  清除缓存数据
- WallePackage.py -d  清除缓存数据以及源APK 
- WallePackage.py -p  自动打渠道包
- WallePackage.py -e  apk_path 反编apk到extract目录下

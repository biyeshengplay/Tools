### 如何使用：
命令行进入该目录->
- AutoPackage.py -h  查看帮助
- AutoPackage.py -c  清除缓存数据
- AutoPackage.py -d  清除缓存数据以及源APK 
- AutoPackage.py -p  自动打渠道包，并且签名
- AutoPackage.py -e package 反编package到extract目录下
